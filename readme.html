はじめに
～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～
これは「誰もがクリエイティブの世界を発展させる灯油（燃料）になるサイト」
をモットーに創られた動画投稿ウェブアプリ、「灯油動画」、
略して「TouYude（とうゆうど）」を目指したものです。
制作期間が極端に短く、制作者のウェブ開発の知識が著しく不足していたため、
完成には至りませんでしたが、
残されたファイルの数々から制作者の熱意が伝わってくるかもしれません。

チュートリアル
～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～
1.「practice01.php」をブラウザで表示する。
2.右上の「ログイン」をクリックする。
3.真下の「新規登録はこちら」にカーソルをゆっくりと重ね、クリックする。
4.「メールアドレス」のテキストボックスに"master"と入力し、中央下部の「新規登録」をクリックする。
5.左下に現れた「元のページに戻る」をクリックする。
6.「メールアドレス」のテキストボックスに"hoge"(任意)と入力し、中央下部の「新規登録」をクリックする。
7.左下に現れた「元のページに戻る」をクリックする。
8.右上の「ログイン」をクリックする。
9.「メールアドレス」のテキストボックスに"hoge"(任意)と入力し、左下部の「ログイン」をクリックする。
10.左下に現れた「元のページに戻る」をクリックする。
11.「メールアドレス」のテキストボックスに"master"と入力し、左下部の「ログイン」をクリックする。
12.左下に現れた「元のページに戻る」をクリックする。
13.これまでに移動してきた各ページで、左上の赤い灯油のアイコンをクリックする。

各ファイルの内容説明
～～～～～～～～～～～～～～～～～～～～～～～～～～～～～～
・practice01.php:
トップページを表示するファイル。

・practice02.css:
全てのページに対して、体裁を整えるファイル。

・practice03.php:
ログインページを表示するファイル。

・practice04.php:
ログイン失敗時に現れるページを表示するファイル。
ログインページで、登録されていないメールアドレスが送信されるとログインページへ、
登録されたメールアドレスが送信されるとトップページへ移動させるリンクを表示する。

・practice045.php:
アカウント登録失敗時に現れるページを表示されるファイル。
新規アカウント登録ページで、すでに登録されているメールアドレスが送信されると
新規アカウント登録ページへ、登録されていないメールアドレスが送信されるとトップページへ
移動させるリンクを表示する。

・practice05.php:
ログイン、新規登録などに使われる関数などを定義するファイル

・practice06.php:
新規会員登録ページを表示するファイル。

・practice08.js:
「新規登録はこちら」を光らせるためのファイル。
アイコンにマウスを重ねたとき、アイコンの灯油タンクから灯油が漏れ、タイトルロゴに着火し、
タイトルロゴのデザインが格好良くなるアニメーションを実装したかった。

・logo.png:
左上に表示するロゴとして使われる画像ファイル。
